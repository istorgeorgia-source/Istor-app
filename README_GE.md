# ISTOR პროფესიონალური Android აპის პროექტი

ეს არის `istor.ge`-სთვის მომზადებული Android Studio / GitHub Actions პროექტი.

## რა აქვს
- აპის სახელი: **ISTOR**
- ქვედა მენიუ: **მთავარი / უკან / განახლება**
- ბრენდირებული splash screen
- WebView ოპტიმიზაცია
- ფაილის ატვირთვა
- ფაილის ჩამოტვირთვა
- გარე ბმულების ბრაუზერში გახსნა
- GitHub Actions-ით APK ბილდი

## APK-ის მიღება GitHub-ზე
1. ატვირთე პროექტის შიგთავსი GitHub repository-ში
2. შედი **Actions**
3. გაუშვი **Build Android APK**
4. დასრულების შემდეგ ჩამოტვირთე artifact
5. შიგნით იქნება `app-debug.apk`

## შენიშვნა
აქ პირდაპირ მზა `.apk` არ დევს, რადგან მის ასაწყობად Android SDK/build tools საჭიროა.
